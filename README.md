## Issues Faced & Solutions
- Issue: 'git' not recognized - Solution: Installed Git for Windows from git-scm.com, chose "Git from command line" option, restarted Cursor.
- Issue: Couldn't locate primary sidebar in Cursor. - Solution : Referred to a youtube tutorial and fixed the issue (View > Appearance > Primary Sidebar)
- I got a Git identity error asking me to configure `user.name` and `user.email`, so I ran:
  `git config --global user.name "Zaid Asif"`
  and
  `git config --global user.email "your-email@example.com"`
  then retried the commit.
